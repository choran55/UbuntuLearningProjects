# UbuntuLearningProjects
POE
 Ubuntu Learning Project Website

This is a simple static website built using HTML, CSS, and a bit of JavaScript.  
It is designed to demonstrate a structured, mobile-responsive, and accessible web design for educational purposes — part of the Ubuntu Learning Project.


 Features

 Responsive layout for desktop and mobile
-  Navigation bar with links to all main pages
-  Home, About Us, and Contact sections
-  Font Awesome integration for social and UI icons
-  Google Fonts for modern typography
-  Contact form (frontend only)
-  Embedded Google Map for location
-  Team member showcase with photos
-  Screenshots of responsive image of the website in different media queries
